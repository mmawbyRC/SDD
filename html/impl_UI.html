Factors affecting readability

Readability is the ease with which a user can read and understand the written work.

Measures are mostly based on the length of words, sentences and paragraphs. However, in terms of user interface, these gives limited assistance.

The use of interface aims to achieve intuitive understanding as an instant. Users should be able to deduce the purpose of each screen element and its realtionship to other elements around it by intuition.
