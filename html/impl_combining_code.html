Methods of Combining Code

Copy-Paste

The easiest method of combining code is copying and pasting the code from the operating systems clipboard. This method only works however when the code is written in the same language and version.

Methods of integrating code

Adjusting Terminology/Lexical Translation

When copying and pasting code from a source where the code is written in a different language or version, changes need to be made to the code, to ensure that the code will work. These adjustments could consist of changing terminology or making grammatical changes, to ensure the code complies with the lexicon.

Importing Files

Modern programming environments allow external modules of code to be incorporated into the source code. Source code can be stored in single library files, that can then be compiled and included in the projects executable files. External modules can be stored in dynamic link libraries, objects or new controls.

Considerations when combining code

Intellectual property rights of author

In all cases, it is unethical to use code created by others without ensuring all intellectual property rights are upheld. This can be done through a a copyright notice, that acknowledges that the code has been created by someone else, and the new application complies with an other licensing agreements.
