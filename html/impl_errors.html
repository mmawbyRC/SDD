Errors

Overview

Syntax Errors

It is an error in the source code of the program, when the source code statements do not adhere to the rules of the specific programming language you are programming in.

These errors are caught during the translation of the code from source code to object code.

A program will not complile until all syntax erros are corrected. In interpreted languages, syntax erros can be detected in program execution.

Logic Errors

Logic errors are mistakes made by the programmer which causes the program to not operate in the intended way, but not crash.

It is hard to catch logic errors as there is nothing ‘wrong’ with the code, there is something wrong with the logic of the code.

Logic errors can be a problem in the processing done, meaning that the output will not be correct.

As the incorrect output is the most clear sign of a logic error, they can be very hard to catch, but using debugging techniques, you can find and correct them.
